# Duncan Lynch

Computer Science student focused on backend systems, low-level infrastructure, and performance-critical C++ development.

Research Assistant @ Stevens Institute of Technology's H2 Design Lab <br>
Course Assistant @ CS-492 (Operating Systems) <br>
Developer behind [Annoloom](https://researchwith.stevens.edu/en/publications/annoloom-augmenting-codebook-generation-and-annotation-with-large/)

## Projects

**Express C** — Single-threaded C HTTP/1.1 request engine.

**UN1T Auth Service** — Stateless JWT-based authentication microservice in Go.

**UN1T Scheduler** — A task scheduler written in Rust that uses DynamoDB and AWS SQS.

---

## [Micro Projects](https://github.com/DuncanLynch/Impl_Practice) (C++)

**Thread Pool** — Fixed-size worker pool with synchronized task queue.

**Arena Allocator** — Linear allocator optimized for locality and bulk deallocation.

**LRU Cache** — O(1) hash map + linked list eviction design.

**Mersenne Twister** — MT19937 implementation.
